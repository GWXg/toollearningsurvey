# A Survey on Tool Learning with LLMs: Frameworks, Components, Paradigms and Strategies

### 综述概述
随着大型语言模型（LLMs）与外部工具的结合成为扩展模型能力的核心方向，“LLM工具学习”领域快速涌现了大量研究，但当前存在**资源碎片化、技术脉络不清晰、范式分类不统一**的痛点。

本综述及配套仓库的核心价值在于：
- 解决领域碎片化问题：搭建“人类-LLM-工具-环境”的统一框架，明确工具学习的核心组件与交互逻辑；
- 系统梳理技术脉络：分类整合工具调用范式、自适应微调策略、评估基准等关键模块的研究进展；
- 提供可复用资源：配套结构化CSV表格、可视化图表，帮助研究者快速定位目标论文、复用基准数据集、理清领域演进路线。

无论你是刚入门的研究者，还是需要梳理领域进展的从业者，本综述及仓库都能帮你高效掌握LLM工具学习的核心内容。
